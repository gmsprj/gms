# GMS

GMS is Guilds Management System.

## Develope

* Language: PHP7
* Frame Work: CakePHP3, AngularJS1.5

## State of project

None.

## Install

### Basic way

* Clone latest GMS repository.
* Create database at your machine.
* Initialize database by ``` gms/ ``` tools.
* Run develop server of CakePHP3
